JuliaFormatter.jl

Width-sensitive formatter for Julia code, inspired by gofmt, refmt, black, and prettier. Built with JuliaSyntax.jl.

  • Provides sane defaults out of the box, with a number of customisation options.

  • Supports YAS, Blue, and SciML style guides.

  • Admits a .JuliaFormatter.toml configuration file for project-level formatting settings.

  • Can be used as a command-line app.

Installation

]add JuliaFormatter

Quick Start

julia> using JuliaFormatter

# Recursively formats all Julia files in the current directory
julia> format(".")

# Formats an individual file
julia> format_file("foo.jl")

# Formats a string (contents of a Julia file)
julia> format_text(str)
